DP로 풀면 시간초과뜸.. Pypy로는 풀리긴 했다

```py
import sys

# 입력
input = sys.stdin.readline
N = int(input())

# 탐색
table = [4] * (N + 1)

for i in range(1, int(N ** 0.5) + 1):
    table[i ** 2] = 1

for n in range(2, N + 1):
    if table[n] == 1:
        continue

    for i in range(1, int((n // 2) ** 0.5) + 1):
        table[n] = min(table[n], table[n - i ** 2] + 1)

        if table[n] == 2:
            break

print(table[N])

```
  
DFS 로 푸니까 잘 풀린다
```py
import sys

# 입력
input = sys.stdin.readline
N = int(input())

# 탐색
answer = [4]
squares = []

for i in range(1, int(N ** 0.5) + 1):
    squares.append(i ** 2)


def dfs(depth, total, start):
    if 0 < depth:
        if total > N:
            return
        if total == N:
            answer[0] = min(answer[0], depth)
        if depth == 3:
            return

    for i in range(start, len(squares)):
        s = squares[i]
        dfs(depth + 1, total + s, i)


dfs(0, 0, 0)
print(answer[0])

```
